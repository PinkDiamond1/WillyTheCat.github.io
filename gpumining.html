How to mine on multiple GPUs

You can start the mining on multiple GPUs with the following RPC commands. If you are using the GUI version of the wallet you can enter the RPC commands under Help/Debug/Console. If you are using the command line version you can use "bitcash-cli xxx" to execute the command xxx.

Do NOT click on the start mining button in the GUI version.

To start mining with multiple GPUs: 

setgenerate true false [true for GPU mining] [ID of first GPU] 1 [number of GPUs to use]

For 2 GPUs:
setgenerate true false true 0 1 2
For 4 GPUs:
setgenerate true false true 0 1 4
For 6 GPUs:
setgenerate true false true 0 1 6
The last number is the number of GPUs to use.

If mine on multiple CPUs you can use this command: 

setgenerate true false false
To see how much mining power you have you can use: 
bitcash-cli getmininginfo
