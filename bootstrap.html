Bootstrap

Download bootstrap file

If you download this file and execute it under Windows it will delete the existing blockchain files (if there are any) and copy the blockchain files from today into your data folder.

It will leave your file wallet.dat untouched. So your existing coins will be okay after executing if. But it can not hurt to make backup. If you are not under Windows you can use RAR to extract the file and copy the files manually into your data folder.

This is a self extracting RAR file which calls a BAT file to delete the old files and copy the new files over.

You need to close the BitCash wallet before you execute the file. After you executed the file you can start your BitCash Wallet and it should have all blocks up to about block 368000.

So you can use this to skip the syncing process if you are just starting a new wallet or you can use it, if you have a syncing problem or any other problem with corrupted files in your data folder to speed up the resyncing process.
