The Windows mining software

Download bitcash-minerd.exe for Windows

Simply download the EXE file and copy it into a folder of your choice for installation. You may need to install the CUDA Toolkit 9.2 for Nvidia GPU mining. You need a NVIDIA graphics card with at least 4 GB of memory. Or you can use CPU mining. You may need to install the Visual Studio 2015 Runtime Libraries if not already installed.

Ubuntu 18.04

Download bitcash-minerd for Ubuntu 18.04

Simply download the file and copy it into a folder of your choice for installation. Set the file permission so that you can execute the file: 

chmod 777 bitcash-minerd
You may need to install the CUDA Toolkit 9.1 for Nvidia GPU mining. You need a NVIDIA graphics card with at least 4 GB of memory. Or you can use CPU mining.
